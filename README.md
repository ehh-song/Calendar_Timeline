# Calendar Todo Widget

날짜별 할일 관리와 하루 일정 타임라인을 한 곳에서 관리하는 데스크톱 위젯입니다.

---

## 실행

```bash
npm install
npm start
```

> 배포용 패키지 생성: `npm run build` → `dist/Calendar-Todo-win.zip`

---

## 화면 구성

```
[ 달력 ] [ 타임라인 ]   ← 탭 전환
```

상단 탭으로 달력과 타임라인을 전환합니다.

---

## 달력 탭

오늘 기준 전후 날짜를 세로로 나열하고, 각 날짜에 할일을 관리합니다.

### 할일 추가 / 완료 / 삭제

| 동작 | 방법 |
|------|------|
| 추가 | 날짜 옆 `+ 추가` 클릭 또는 날짜 선택 후 `Enter` |
| 완료 토글 | 항목 클릭, `Space`, 또는 `Enter` |
| 삭제 | 항목 선택 후 `Delete` / `Backspace`, 또는 우클릭 → 삭제 |
| 수정 | 항목 선택 후 `F2`, 또는 우클릭 → 수정 |

완료된 항목은 자동으로 아래로 정렬됩니다.

### 할일 이동

- **드래그 앤 드롭**: 항목을 다른 날짜로 끌어다 놓기
- **잘라내기/붙여넣기**: 항목 선택 → `Ctrl+X` → 다른 날짜 선택 → `Ctrl+V`

이동 시 완료 상태는 초기화됩니다.

### 키보드 탐색

| 키 | 동작 |
|----|------|
| `↑` / `↓` | 항목 이동 (날짜 경계 넘어서도 연속) |
| `Tab` / `Shift+Tab` | 날짜 섹션 이동 |
| `Esc` | 선택 해제 / 잘라내기 취소 |

### 컴팩트 모드

우상단 `⬜` 버튼 또는 `Ctrl+M` — 오늘 날짜만 표시, 창이 자동으로 작아집니다.

## 타임라인 탭

하루를 시간표 형태로 보여주며, 시간 블록 단위로 일정을 관리합니다.

### 일정 추가

빈 영역을 **클릭 후 드래그**하면 시간 범위를 선택할 수 있습니다 (최소 15분).  
드래그를 놓으면 일정 추가 모달이 열립니다.

- 이름, 시작/종료 시간, 카테고리 선택 후 저장

### 일정 수정 / 삭제

- **우클릭** → 수정 모달 열기
- 모달 하단 `삭제` 버튼으로 삭제

### 일정 이동

일정 블록을 **드래그**하면 같은 날 다른 시간으로 이동합니다 (10분 단위 스냅).

### 노트 작성

일정 블록을 **좌클릭**하면 노트 모달이 열립니다.

- **잘한 점 / 아쉬운 점 / 깨달은 점** 세 항목으로 회고 기록 가능

### 날짜 이동

상단 `‹` `›` 버튼으로 날짜를 이동합니다. 오늘이 아닌 날에는 `오늘` 버튼이 나타납니다.

### 카테고리

일정 모달의 `관리` 버튼으로 카테고리 색상/이름을 추가·수정·삭제할 수 있습니다.  
기본 카테고리: 업무, 개인, 미팅, 기타

---

## 창 관리

| 버튼 | 기능 |
|------|------|
| 📌 (핀) | 항상 위에 고정 |
| `⬜` | 컴팩트 모드 전환 |
| `⚙` | 설정 (단축키 커스터마이징) |
| `−` | 최소화 |
| `✕` | 닫기 |

창 위치와 크기는 앱 종료 후에도 저장됩니다.

---

## 단축키 커스터마이징

설정(`⚙`) 패널에서 8개 액션에 원하는 단축키를 지정할 수 있습니다.  
배지를 클릭 후 원하는 키 조합을 입력하면 등록됩니다.

| 액션 | 기본 단축키 |
|------|------------|
| 할일 추가 | `Ctrl+N` |
| 잘라내기 | `Ctrl+X` |
| 붙여넣기 | `Ctrl+V` |
| 컴팩트 전환 | `Ctrl+M` |
| 탭 전환 / 핀 / 최소화 / 닫기 | 미설정 |
